Proyectos realizados


App de Notas

Proyecto de Notas que funciona de forma Online y Offline con Service Worker
Se puede instalar (manifest.json) la aplicación y utilizarlo de forma desktop como mobile. Anota lo que quieras en tiempo real con fecha y hora exacta.

Buscador de peliculas

Buscador de películas trabajado con Vue.js. Renderiza de forma rápida y guarda la selección
en el localStorage. Se puede likear al card de tu película favorita.

Tienda e-Commerce

Tienda virtual realizada como trabajo final de la carrera de Diseño Web y Programación.
Se llevaron a cabo las metologias ágiles y la buena comunicación entre los compañeros para tener una mayor organización, así como también la
